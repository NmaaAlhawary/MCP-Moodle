Quick start

Install once.

Two steps: the Moodle plugin, then the MCP server.

bash
$ git clone https://github.com/NmaaAlhawary/MCP-Moodle $ cd MCP-Moodle/moodle-mcp $ python3 -m venv .venv && source .venv/bin/activate $ pip install -r requirements.txt $ cp .env.example .env # add MOODLE_URL + token $ python server.py

Full guide in the docs.
