Setup

Four steps to get running

Each step below has a slot for a short video clip. Drop your recording into the videos/ folder to embed it.

1

Install the plugin

In Moodle: Site administration → Plugins → Install plugins, upload mcpbridge.zip from the latest release, and confirm the database upgrade. The MCP Bridge plugin appears under Local plugins.

videos/01-install.mp4
2

Enable web services & create a token

Turn on web services and the REST protocol, add the functions to the MCP Bridge Service, authorise a user, then create a token under Manage tokens. Copy it — the AI uses it to talk to Moodle.

videos/02-token.mp4
3

Connect your AI client

Run the Python MCP server with your MOODLE_URL and MOODLE_TOKEN, then add it to claude_desktop_config.json. Restart Claude Desktop — a moodle connector appears, toggled on.

videos/03-connect.mp4
4

Start building — just chat

Type what you want: “Create a course called Biology 101 and add a welcome page.” The AI calls the right tools and does it live. That's it.

videos/04-build.mp4
💡 Adding videos: put your screen recordings in docs/videos/ (e.g. 01-install.mp4), then replace a video slot's markup with a <video controls> tag — see the comments in docs/index.html. You can also embed YouTube links.
Inside Moodle

What it can actually do

Everything below happens directly in your Moodle site — real records, not mock-ups. Each card shows an example of what you'd say.

🏫

Create courses & categories

Spin up single courses or entire programs, organised into categories.

“Create a Web Dev Bootcamp category with 4 courses.”
📄

Add pages & content

Create Page activities with rich HTML content, section by section.

“Add a Week 1 page with a welcome message to course 7.”
📚

Build books & resources

Multi-chapter Books, Labels, and URL resources for reading material.

“Create a book called ‘Study Guide’ with 3 chapters.”
📝

Set up quizzes

Create quiz activities and add multiple-choice questions to them.

“Add a 5-question multiple-choice quiz on cell biology.”
👥

Manage people

Create users, enrol them as students or teachers, and build groups.

“Enrol these five students in Biology 101 as students.”
📊

Read insights

Grades, deadlines, progress, and combined dashboards across courses.

“What are my upcoming deadlines and overdue tasks?”
Try these

Copy-paste prompts

Paste any of these into your AI client once the moodle connector is on.

›“Verify my Moodle connection and list all my courses.”
›“Create a course called ‘Intro to Python’ and add three pages: Variables, Loops, and Functions — each with a short explanation.”
›“In course 7, build a book called ‘Course Handbook’ with chapters for Syllabus, Grading, and Schedule.”
›“Create an empty quiz called ‘Week 1 Check’ in course 9, then add a multiple-choice question about JavaScript variables.”
›“Enrol me as a teacher in every course in the Web Development Bootcamp category.”
›“Give me a semester dashboard: courses, upcoming deadlines, and recent grades.”
