Tool reference

87 tools the AI can call. Read tools are always available; write tools require MOODLE_ALLOW_WRITE=true.

Courses & structure

create_course · create_category · create_section · update_course · delete_section · delete_courses · delete_category · import_course · get_course_content · list_courses · search_courses · get_my_courses

Course builder & backup

build_course builds a whole course — sections, activities, quizzes and their questions — from one JSON outline. backup_course saves a real Moodle .mbz backup to your local disk; restore_course restores it into a brand-new course.

Content & activities

create_page · create_book · create_label · create_url · create_forum · create_choice · create_assignment · update_page · update_activity · move_activity · delete_activity · upload_file · download_file

Quizzes & questions

create_quiz · add_quiz_question · add_truefalse_question · add_shortanswer_question · add_essay_question · add_numerical_question · add_matching_question · list_quizzes · get_quiz_results · get_quiz_attempt_review

Six question types: multiple choice, true/false, short answer, essay (manually graded), numerical with tolerances, and matching pairs. get_quiz_results shows a user's attempts and best grade; get_quiz_attempt_review walks a finished attempt question by question.

Chatbot bridge: ask_course_chatbot answers from the same RAG content index as the block_chatbot in-page assistant. It needs that block installed, enabled and synced for the course — otherwise it returns a clean error.
